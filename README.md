# Шпаргалка — Билеты

Простой сайт-шпаргалка: 30 кнопок (по номерам билетов), при нажатии открывается
PDF на нужной странице, и заголовок «БИЛЕТ №N» подсвечивается на ~3 секунды
(как поиск в Telegram).

## Файлы

- `index.html` — весь сайт (один файл, использует PDF.js с CDN)
- `tickets.pdf` — твой PDF с билетами

## Как опубликовать на GitHub Pages

1. Создай новый репозиторий на GitHub (например `bilety-shpargalka`).
2. Залей туда оба файла — `index.html` и `tickets.pdf` — в корень репозитория.
3. В репозитории зайди в **Settings → Pages**.
4. В разделе **Build and deployment** выбери Source: **Deploy from a branch**,
   branch: **main**, папка: **/ (root)**. Сохрани.
5. Через 1-2 минуты сайт будет доступен по адресу вида:
   `https://<твой-логин>.github.io/bilety-shpargalka/`

Готово — открывай ссылку на телефоне или компьютере и жми на номера билетов.

## Если захочешь поменять номера страниц

В `index.html` есть массив:

```js
const TICKET_PAGES = [1,4,7,10,14,19,24,27,31,33,39,43,45,48,50,54,59,62,65,70,75,80,85,91,95,99,104,107,111,116];
```

Это страница, на которой начинается каждый билет (билет 1 → страница 1,
билет 2 → страница 4, и т.д.). Если заменишь PDF на другой — пересчитай эти
номера.
